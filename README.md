# 🧮 Calculadora com Electron

Uma calculadora desktop simples desenvolvida utilizando **JavaScript puro**, **HTML/CSS** e o **Electron**.

## 🚀 Funcionalidades

* Operações básicas (+, -, *, /)
* Suporte a entrada pelo teclado
* Interface simples e intuitiva
* Execução como aplicativo desktop
* Tratamento básico de erros

## 🛠️ Tecnologias

* JavaScript (Vanilla)
* HTML5
* CSS3
* Electron

## ▶️ Como executar o projeto

```bash
# Instalar dependências
npm install

# Rodar o projeto
npm start
```
## 📦 Download

Você pode baixar a versão executável (.exe) na aba de Releases do repositório.

## 📌 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos de:

* Manipulação do DOM
* Eventos de teclado
* Estruturação de aplicações desktop com Electron
* Organização de projeto frontend

## 💡 Melhorias futuras

* Interface moderna (UI/UX aprimorada)
* Histórico de cálculos
* Substituição do eval() por uma abordagem mais segura
* Melhor tratamento de erros
