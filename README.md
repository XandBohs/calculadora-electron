# 🧮 Calculadora com Electron

Uma calculadora desktop simples desenvolvida utilizando **JavaScript puro**, **HTML/CSS** e o **Electron**.

## 🚀 Funcionalidades

* Operações básicas (+, -, *, /)
* Suporte a entrada pelo teclado
* Interface simples e intuitiva
* Execução como aplicativo desktop

## 🛠️ Tecnologias

* JavaScript (Vanilla)
* HTML5
* CSS3
* Electron

## ▶️ Como executar o projeto

```bash
# Instalar dependências
npm install

# Rodar o projeto
npm start
```

## 📌 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos de:

* Manipulação do DOM
* Eventos de teclado
* Estruturação de aplicações desktop com Electron

## 💡 Melhorias futuras

* Interface mais moderna (UI/UX)
* Histórico de cálculos
* Suporte a mais operações matemáticas
* Empacotamento em .exe
